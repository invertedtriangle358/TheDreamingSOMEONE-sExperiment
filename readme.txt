-夢みる"だれかさん"の実験-



だれかさんは手鏡を探している。どくろさんはそれをみている

私は母親を殺した。手鏡なんて隠していない。どくろさんはそれをみていない。

手鏡はどこにもない。死体は消えた。どくろさんは答えない。



だれかさんは狂っていない。赤と黒の悪夢から目覚めない。

これは誰の夢なのか。





私あるいは貴方、彼あるいは彼女はだれなのか。
